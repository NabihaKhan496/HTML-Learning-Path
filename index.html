HTML Document Structure

Every webpage starts with a basic HTML structure. HTML uses elements to organize content and give meaning to a webpage.

<!DOCTYPE html>

HTML5 Declaration

Tells the browser that the document uses modern HTML5.

<html>

Root Element

Contains every other element in the webpage.

<head>

Page Information

Contains metadata, title, styles, and linked resources.

<body>

Visible Content

Contains everything users see and interact with.

⚠️ Deprecated HTML Tags

These elements are useful to recognize in older websites, but modern projects should use CSS and semantic HTML instead.

<font>

Use CSS Instead

Old way of changing font color and size.

<center>

Use text-align

Modern CSS uses text-align: center.

<marquee>

Use CSS Animation

Scrolling text should be created with modern CSS.

<tt>

Use <code>

Use semantic code elements instead.

<strike>

Use <del>

Use the semantic deletion element.

<acronym>

Use <abbr>

Use abbreviation semantics.
